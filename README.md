# 💸 Costify

**Costify** is a SwiftUI-based personal expense tracking app that helps you manage your daily expenses with elegant visualizations and intuitive UI. Built with Core Data for persistent local storage, Costify features a custom SwiftUI pie/donut chart—no external charting libraries needed.

---

## 🚀 Features

- **➕ Add / ✏️ Edit / 🗑️ Delete** expenses with:
  - Title
  - Amount
  - Category
  - Date
  - Notes

- **📅 Filter by Date:**
  - Today
  - Last 7 Days
  - Last 14 Days
  - Last 30 Days
  - Last 1 Year

- **📊 Interactive Pie (Donut) Chart**
  - Visualizes category-wise spending
  - Custom-built in SwiftUI—no third-party chart packages

- **🎨 Color-coded Categories**
  - Each category has a distinct color and icon (SF Symbol)
  - Clean legend alongside the chart

- **🏠 Compact HomeView**
  - Summary, chart, and categorized expense list
  - Light & Dark mode support

- **📁 Persistent Core Data Storage**
  - Manual `NSManagedObject` subclassing (no auto-codegen)
